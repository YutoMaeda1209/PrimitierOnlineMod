# PrimitierOnlineMod

Mod to add multiplayer functionality to Primitier.

## Description

This mod adds multiplayer capabilities to the VR game Primitier.

## Getting Started

### Dependencies

* [.NET 6.0](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
* [MelonLoader 0.6.2](https://github.com/LavaGang/MelonLoader)

### Installing

* Install [MelonLoader 0.6.2](https://github.com/LavaGang/MelonLoader) into Primitier according to the installer.
* Download the latest version of mod data from [the release page of this repository](https://github.com/YutoMaeda1209/PrimitierOnlineMod/releases) and extract it in the Primitier/Mods folder.
